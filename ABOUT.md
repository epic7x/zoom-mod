# Zoom

<cg>Zoom</c> lets you inspect a level while the game is paused by adding smooth camera zoom and movement controls.

This fork is maintained by <cy>epic7x</c> for modern <cl>Geode</c> and <cl>Geometry Dash 2.2</c> builds.

<co>This mod is open source.</c> The source code is available for review, and it does <cr>not</c> include malicious behavior, hidden network activity, or intentionally harmful code.

# What It Does

- <cg>Zoom in and out</c> while paused
- <cy>Move the camera</c> around the level
- <cl>Show or hide</c> the pause menu while zoomed
- Use <cp>touch gestures</c> on mobile for easier inspection

The goal is to make paused level inspection feel clean, practical, and comfortable on both desktop and mobile.

# Controls

## Desktop

- Use the <cg>Scroll Wheel</c> to zoom in or out
- Hold <cy>Middle Mouse Button</c> to move the camera
- Press <cl>Home</c> to toggle the pause menu

The keybind can be changed from the mod settings in Geode.

## Mobile

- Open the pause menu
- Tap the <cy>Magnifying Glass</c> button to enter <cl>Zoom Mode</c>
- <cg>Pinch to zoom</c>
- <cy>Drag with one finger</c> to move the camera
- Use the <cl>back button</c> in the top-left corner to exit

# Settings

- <cg>Zoom Rate</c>: changes desktop zoom speed
- <cy>Hide Menu Automatically</c>: hides the pause menu when zooming in
- <cy>Show Menu Automatically</c>: shows the pause menu again when zooming out
- <cl>Alt Key Stops Zoom</c>: disables zoom input while <cl>Alt</c> is held
- <cp>Toggle Pause Screen</c>: desktop keybind for showing or hiding the pause menu

# Compatibility

- <cl>Geode</c>: <cg>5.5.3</c>
- <cl>Geometry Dash</c>: <cg>2.2081</c>
- <cl>Platforms</c>: <cg>Windows</c>, <cg>Android</c>, <cg>macOS</c>, and <cg>iOS</c>

Desktop support is currently the most polished. Apple platforms are supported, but may still need more testing on real devices.
