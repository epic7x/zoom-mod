# Zoom
Enables zooming while the game is paused.

# Controls
## Desktop
- Use the <cg>Scroll Wheel</cg> to zoom in or out
- Hold <cg>Middle Mouse Button</cg> to move the camera
- Press <cl>Home</c> to toggle the pause menu
## Mobile
- Tap the <cy>Magnifying Glass</c> icon in the pause menu to activate <cl>zoom mode</c>
- While in <cl>zoom mode </c>, you can <cg>pinch to zoom</c>, and exit by pressing the back button located at the top left

# macOS and iOS
- Supports macOS and iOS, but it hasn’t been fully tested yet; a complete review will be carried out soon.
